# prompts

Templates texte utilises pour les appels LLM (headers, procedures, SQL, diagnostic/repair).

Fichiers :
- `header_template.txt`, `procedure_template.txt`, `sql_template.txt` : prompts principaux pour generer COBOL/SQL.
- `diagnosis_template.txt`, `repair_template.txt`, `compiler_test_template.txt` : prompts de debug/validation.

Modifier ces fichiers pour ajuster le ton, le contexte ou les consignes donnees au modele LLM.
